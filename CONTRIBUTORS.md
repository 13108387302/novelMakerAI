## 🤝 如何成为贡献者

### 1. 代码贡献
```bash
# Fork项目
git clone https://github.com/your-username/ai-novel-editor.git

# 创建分支
git checkout -b feature/your-feature

# 提交代码
git commit -m "feat: add your feature"

# 创建Pull Request
```

- 改进现有文档
- 添加新的教程

### 3. 社区贡献

- 推广项目
- 组织活动

## 🏅 贡献者徽章

根据贡献程度，我们为贡献者提供不同的徽章：

- 🥇 **金牌贡献者**: 重大功能开发或长期贡献
- 🥈 **银牌贡献者**: 重要功能改进或多次贡献
- 🥉 **铜牌贡献者**: 有价值的贡献或首次贡献
- ⭐ **特殊贡献者**: 在特定领域有突出贡献
